Resouce Aid Form Browser Guidance

Please read the instructions below before starting to complete the form.

  • For Smart Phone Users:

    You need to download free "Acrobat Reader DC mobile app" on your device to be able to edit the form, you can edit the form and email it using this app.

  • For Windows 10 Users

    Google Chrome Browser:

    We don't advise you to use Google Chrome, as it's not compitable with Acrobat.

    Microsoft Edge Browser:

    Windows 10 comes with Microsoft Edge Browser, this browser is still in developmental stage, the Resource Form will open disabled on Microsoft Edge, you can't edit it, we advise you to use Internet Explorer 11, the form will work proberly. If you don't have it, you can download it from Download IE 11

    Mozilla Firefox Browser:

    1. Choose Options from the menu, then choose Applications from the side bar.
    2. Select Portable Document Format (PDF) from the Content Type Column.
    3. Click the Action column next to Portable Document Format (PDF), and then select Use Adobe Acrobat In FireFox.

To open the application form, click on the link: Resource Aid Form.
